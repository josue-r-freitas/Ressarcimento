Sistema de Ressarcimento ICMS-ST – SEFAZ/AM

API REST para geração de XMLs de ressarcimento de ICMS por substituição tributária (produtos e pedidos), conforme manual da SEFAZ/AM e envio via DT-e.

Declarante

  • POST /api/declarante – Cadastrar ou atualizar dados do declarante (body JSON)
  • GET /api/declarante – Consultar dados do declarante

Produtos (MATRI-NAC)

  • POST /api/produtos/importar – Upload da planilha de produtos (multipart/form-data, campo arquivo)
  • GET /api/produtos – Listar produtos (query opcional: codigo, descricao, page, size)
  • POST /api/produtos/gerar-xml – Gerar XML enviProdutoRessarcimento e retornar para download
  • GET /api/produtos/historico – Listar histórico de arquivos de produtos gerados
  • GET /api/produtos/historico/{id}/download – Download do XML de produtos pelo ID do histórico

Pedidos (versão 2.00)

  • POST /api/pedidos/importar – Upload da planilha de operações de saída (multipart/form-data, campo arquivo)
  • GET /api/pedidos – Listar operações/notas de saída (query opcional: ano, mes, page, size)
  • POST /api/pedidos/gerar-xml – Gerar XML enviOperacaoRessarcimento (query obrigatória: ano, mes)
  • GET /api/pedidos/historico – Listar histórico de arquivos de pedidos gerados
  • GET /api/pedidos/historico/{id}/download – Download do XML de pedidos pelo ID do histórico
